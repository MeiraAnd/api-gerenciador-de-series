API em Node.js
==============

O que é isso?
============

API em Node.js tem como objetivo gereciar séries na plataforma Netflix.

Como foi feito?
===============

- NodeJs.
- Axios.
- MongoDB (Mongoose).
- Express.

Instalação:
==========

1. Instale o Node.js. Depois digite no terminal npm install.
2. Clone o repositório.
3. Cd /nome-do-repositorio.
4. node index.js.
